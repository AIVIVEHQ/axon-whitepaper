# Part VI · Roadmap & Governance

> Four stages, each with exit value; L1 + PayFi first, TradFi is the long-range vision.

A grand vision, without a pragmatic, staged path to delivery, is just fantasy. This part lays out AXON's cadence of progress from testnet to a global asset network, along with the governance framework that keeps the chain running over the long term — and the people behind it.

The design principle of AXON's roadmap is **"every stage has exit value"** — each stage is not a transition "for the sake of reaching the next," but delivers usable, verifiable value in its own right. This gives the whole path both a far-reaching destination and solid waypoints along the way.

Chapter map for this part:

* **6.1** Roadmap P0 → P3+ — a four-stage advance from testnet to a global asset network.
* **6.2** Governance Framework — the path from a centralized launch to decentralized governance taking over.
* **6.3** Team & Resource Network — the people building this chain, and the network supporting it.
* **6.4** Ecosystem Launch & Season 1 — recruiting project teams to onboard via an ecosystem open program, trading token incentives for real growth.
