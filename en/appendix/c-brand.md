# Appendix C · Brand & Trademark

This appendix excerpts the core guidelines of the AXON Finance visual identity (VI), for use in typesetting this whitepaper and derivative materials. The official VI manual is the authoritative source for the full guidelines.

Finalized combination: **F0 emblem × P7 Obsidian · Ember × W2 circuit wordmark**.

## Color Tokens (P7 Obsidian · Ember)

| Role | Name | HEX | Use |
| --- | --- | --- | --- |
| Primary | Ember | `#f5541c` | The brand's sole accent color, logo primary, CTA |
| Deep base | Obsidian | `#0a0908` | Dark-background base |
| Light base / paper | Warm White | `#f7f4f1` | Light backgrounds, reversed-out text |
| Text | Ink | `#16100b` | Body text on light backgrounds |
| Secondary | Secondary Gray | `#8f8578` | Supporting text, dividers, annotations |

**Gradients (only two — do not invent your own):**

* **G1 Ember Glow** (light scenes): `#f0500f → #ff8a3d → #ffe9d6` — light-scene cards, onboarding pages, poster backgrounds.
* **G2 Dark Glow** (dark-scene radial): `#ffb073 → #f5541c → #5a1f06 → #0a0908` — dark hero sections, data-visualization backgrounds.

> Color discipline: Ember is the **only** high-saturation accent color; avoid large high-saturation color blocks; introduce warm accents as rim light / glow / gradient.

## Typography System

| Role | Typeface | Notes |
| --- | --- | --- |
| Logotype | **W2 custom wordmark** | Pure geometric vector wordmark, not a font, no need to outline |
| Display | **Bricolage Grotesque** | Headlines, large type |
| Body | **Geist Sans** | Body text, UI |
| Data / Mono | **Geist Mono** | Numbers, code, on-chain data |
| Chinese | **Noto Sans SC / PingFang SC** | Parallel fallback to the above |

## Logo Usage Rules

**Do**: leave ample whitespace; keep the logo larger than the text (horizontal emblem height : text height ≈ 2.9 : 1); use reversed-out / line-art versions on dark backgrounds; match color values to the table above verbatim.

**Don't**: alter the logo's colors; stretch or distort it; add an outline stroke to the solid version; 3D-ify the emblem as the visual centerpiece; use the glass version as the logo itself.
