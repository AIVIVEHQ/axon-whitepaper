# Part IV · PayFi 引擎

> 一条支付 rail 打底，四个场景层层叠加价值。

Part III 建好了地基。本部分展示地基之上生长出的业务：**PayFi 的四大场景**。

这四个场景不是并列的四个功能，而是一个**层层叠加的价值结构**——底层是结算 rail，它承载一切；在其上叠加货币市场，捕获时间价值；再叠加跨境与商户场景，触达实体商业；而 AI 代理支付（Part V）则贯穿其中，成为机器时代的支付方式。
