# 4.5 美股带单引擎：首个开放生态

前四节把 PayFi 的地基场景（结算 rail、货币市场、跨境收单、时间价值）讲清楚了。本节讲**第一个把这些能力兑现成真实业务的落地产品**——**AXON 美股带单引擎（Copy Trading Engine）**。

在 [4.2 PayFi 货币市场](4-2-money-market.md) 描述的「浮存金 + 链上信贷」方向上，带单引擎是 AXON Finance **首个开放生态**——AXON 提供 L1 结算与账户抽象基建，预测平台与硅谷量化交易台作为生态参与方入驻（AXON 是基建，非产品运营方）。它不追求一上来就铺满所有场景，而是选一个**流量、收益、闭环都能被真实验证**的切口先跑通：它的核心使命，是通过为外部流量提供确定性稳定币收益，实现 L1 的 TVL 与真实地址数的爆发式增长。

## 战略赋能：流量互换模型

带单引擎专为 Polymarket、Kalshi、Kairos **等预测平台**的用户设计。它解决的是这些平台上一个长期被忽视的结构性问题：

* **痛点**——预测市场的事件往往周期很长（一场大选、一次财报、一个宏观决议），用户资金在等待结果期间大量闲置，且缺乏跨界的金融工具去盘活这段闲置。
* **解决方案**——预测平台作为入口提供流量，AXON 提供底层的交易与结算基建。用户直接用预测平台上的稳定币，通过 AXON L1 **一键跟单**美股策略，让闲置资金在等待期里持续生息。
* **价值闭环**——AXON 获得庞大的真实流量与 TVL，预测平台获得一个高粘性的生息理财功能。**「预测平台引流，AXON 稳定币生息」**，双方各取所需。

在对外定位上，这一层合作以「**合作生态入驻**」的形式呈现：预测平台是 AXON 生态的合作伙伴、是[第一赛季生态开放活动](../part6-roadmap/6-4-ecosystem-season.md)的入驻项目方之一，而非某种深度独家绑定。生态是开放的，带单引擎只是把这条开放生态的第一批流量接了进来。

## 核心机制：保本保底（1%~3%）

系统向用户承诺「**本金保护 + 确定性单次收益**」，单次收益区间为 **1%~3%**。这个承诺不是靠拆东墙补西墙的资金盘，而是由**四层真实盈亏支柱**共同支撑，从设计上坚决排除庞氏模型：

1. **美股量化边际**——由 AXON 的前硅谷大厂核心团队（见 [6.3 团队与资源网络](../part6-roadmap/6-3-team-partners.md)）执行的高胜率美股量化策略（期权套利、IV 曲面偏离），主要针对英伟达财报、美联储决议等**具有确定性催化剂**的高频事件。
2. **期权对冲**——通过跨式、铁鹰等期权策略，把损失风险**硬锁在已知范围内**，拒绝裸多裸空。收益的确定性，来自风险边界的确定性。
3. **真实营收注水**——AXON L1 链上的网络 Gas 费、PayFi 结算费按固定比例注入准备金池。网络越活跃，兜底的底气越厚。
4. **获客预算让利**——把传统公链投在空投 / 宣发上的获客预算，直接转化为对真实交易用户的「保底补贴」。用户增长的成本，回流给了用户本身。

## 收益分配：抽成注入底池

带单引擎从用户收益中按比例抽成，这部分资金**全部注入 AXON 底池**，用于支撑公链的流动性与代币价值。它与 [7.3 代币效用与通缩飞轮](../part7-tokenomics/7-3-utility-flywheel.md) 的价值捕获机制直接对接：

* 一部分随协议手续费一起进入**回购销毁飞轮**——真实业务量 → 协议营收 → 回购 AXON → 销毁，用真实现金流支撑流动性与币价，而非通胀补贴；
* 一部分补充**链上带单准备金池**，为「保本保底」提供终极兜底。

也就是说，带单引擎每跑通一单真实交易，既让用户拿到确定性收益，也把一部分价值沉淀回 L1 本身。

## 系统架构与安全垫

* 用户资金完全受 AXON L1 **智能合约**管控，配合账户抽象（AA）与 Paymaster 模块（见 [3.7 账户抽象、会话密钥与 Paymaster](../part3-architecture/3-7-account-abstraction.md)）实现**无感、零 Gas** 的交互体验。
* 设立**链上带单准备金池**进行终极兜底。系统实时监控准备金覆盖率（如设定阈值 **≥ 150%**），当覆盖率不足时**智能熔断**——自动限缩场次发布额度，确保在途资金的绝对安全。

带单引擎是 AXON「地基先行、产品切口」战略的第一次兑现：它把 Part III 的确定性地基、Part IV 的 PayFi 收益能力、Part V 的 AI 可控执行拧成一个能被市场直接感知的产品，也为[路线图](../part6-roadmap/6-1-roadmap.md) P1 阶段提供了第一个「有真实用户、真实资金、真实收益」的落地锚点。
